### Instalação e Execução do Projeto

- Clone este repositório

```
> git clone https://github.com/daygorgonha/flutter-todo-list
```

- Navegue até o diretório principal do projeto

```
> cd flutter-todo-list
```

- Instale as dependências

```
flutter doctor
```

- Execute o projeto

```
flutter run
```
